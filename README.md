# WebSocket 广播系统

## 功能说明
- 服务端支持多客户端连接，为每个连接分配唯一ID
- 客户端发送的消息会广播给所有其他客户端
- 客户端断开连接时会通知其他客户端

## 环境要求
- Python 3.7+
- 依赖包: `fastapi`, `uvicorn`, `websockets`

pip install fastapi uvicorn websockets
